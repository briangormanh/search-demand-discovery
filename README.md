# What the Search Box Was Trying to Tell Us

A Sixth City Marketing research & discovery case study: reading 12 months of real, conversational Google Search Console queries for an (anonymized) multi-location pet-boarding brand, clustering them into demand themes, and turning them into on-page actions — including the "board near the airport" insight nobody had named.

Single self-contained static page (`index.html`) deployed via GitHub Pages. Client details anonymized; query data is real.
